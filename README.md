#  Password Strength Checker

A Python application that analyzes password strength using common cybersecurity best practices.

## Features

- Checks password length
- Checks uppercase letters
- Checks lowercase letters
- Checks numbers
- Checks special characters
- Gives a password strength rating

## Technologies

- Python
- Regular Expressions (re)

## How to Run

```bash
py main.py
```

## Example

```
===== PASSWORD STRENGTH CHECKER =====

Enter your password: MyPassword@123

Password Analysis
------------------------------
✅ Very Strong Password
```
